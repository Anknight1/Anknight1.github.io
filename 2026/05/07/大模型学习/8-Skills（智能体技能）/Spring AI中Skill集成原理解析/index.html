总结

Spring AI Alibaba 的 Skills 集成实现了一套清晰的分层架构：SkillRegistry 负责技能的加载和存储，SkillsAgentHook 负责将技能注入 Agent 上下文，read_skill 工具负责按需加载完整内容。整个设计遵循渐进式披露原则，先注入轻量的元数据，再按需加载重量的完整内容，在功能完整性和 Token 效率之间取得了良好的平衡。
