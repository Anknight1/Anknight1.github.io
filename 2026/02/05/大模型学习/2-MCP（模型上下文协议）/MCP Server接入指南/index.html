总结

接入 MCP Server 非常简单：

  1. 找到配置文件：不同客户端配置文件位置不同
  2. 添加 Server 配置：stdio 方式填 command/args，SSE 方式填 url
  3. 重启客户端：配置生效需要重启
  4. 验证工具加载：查看工具图标或日志确认

MCP 生态正在快速发展，建议关注 MCP Server 官方仓库 获取最新的可用 Server 列表。
