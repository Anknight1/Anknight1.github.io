总结

Spring AI 中 MCP 调用的完整链路：

  1. 启动时：McpSyncClient 连接 Server，SyncMcpToolCallbackProvider 获取工具列表并注册
  2. 请求时：ChatClient 将工具定义发给 LLM
  3. LLM 决策：LLM 返回工具调用请求（tool_use）
  4. 调度执行：ToolCallingManager 找到对应 SyncMcpToolCallback 并执行
  5. MCP 通信：McpSyncClient 通过传输层发送 tools/call 请求
  6. 结果整合：工具结果发回 LLM，生成最终回答
