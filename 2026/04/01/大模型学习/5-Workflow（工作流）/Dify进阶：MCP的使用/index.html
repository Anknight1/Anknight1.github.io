Dify 从 0.9.x 版本开始支持 MCP（Model Context Protocol），可以将任意 MCP Server 集成为 Dify 的工具，在工作流和 Agent 中使用。

MCP 在 Dify 中的作用

MCP 让 Dify 能够连接外部工具和服务：
