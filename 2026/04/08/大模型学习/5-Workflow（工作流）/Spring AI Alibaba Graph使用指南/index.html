Spring AI Alibaba 提供了 Graph 模块，让 Java 开发者可以用熟悉的 Spring 生态构建状态图工作流，实现类似 LangGraph 的 Agent 编排能力。

Spring AI Alibaba Graph 简介

Spring AI Alibaba Graph 是基于 Spring AI Alibaba 框架的工作流编排模块，核心概念与 LangGraph 类似：

  • StateGraph：状态图，定义工作流的结构
  • OverAllState：全局状态，在节点间传递数据
  • NodeAction：节点动作，定义每个节点的处理逻辑
  • Checkpointer：检查点，支持状态持久化和 Human-in-the-Loop
