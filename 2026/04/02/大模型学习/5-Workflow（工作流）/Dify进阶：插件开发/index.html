前言

本文是「大模型学习」系列文章之一，持续更新中。

Dify 插件系统允许开发者为 Dify 添加自定义工具、模型和扩展能力。本篇介绍如何开发一个 Dify 插件。

Dify 插件类型

Dify 支持三种插件类型：

Tool（工具插件）：为工作流和 Agent 提供可调用的工具，如搜索、计算、数据处理等。

Model（模型插件）：接入新的 LLM 或 Embedding 模型供应商。

Extension（扩展插件）：扩展 Dify 的其他能力，如自定义存储、认证等。

安装到 Dify

  1. 进入 Dify 管理后台 → 插件
  2. 点击「安装插件」→「本地安装」
  3. 上传 .difypkg 文件

总结
