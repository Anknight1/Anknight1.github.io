总结

Agent 的记忆体系分为短期和长期两个层次：

  • 短期记忆：存储在 Context Window 中，通过 ChatMemory 管理对话历史
  • 长期记忆：存储在外部数据库（关系库、向量库、Redis），跨会话持久化
  • Checkpointer：持久化 Agent 的执行状态，支持断点续传

合理设计记忆体系，可以让 Agent 具备真正的”个性化”能力。
