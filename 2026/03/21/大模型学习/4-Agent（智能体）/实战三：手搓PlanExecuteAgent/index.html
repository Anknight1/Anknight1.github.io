本文手搓一个 Plan and Execute Agent，以技术选型报告生成为场景：给定一个技术选型需求（如”为新项目选择消息队列”），Agent 自动制定调研计划、逐步收集信息、动态调整计划，最终生成一份完整的技术选型报告。

总结

Plan and Execute Agent 的完整实现包含四个核心组件：

TaskPlanner 将目标分解为具体步骤，StepExecutor 逐步执行每个步骤并利用工具获取信息，TaskReplanner 根据执行结果动态调整剩余计划（可提前终止或调整方向），ReportGenerator 汇总所有执行结果生成最终输出。

这种架构特别适合需要多步骤信息收集和综合分析的任务，如技术选型、竞品分析、技术评估等。与 ReAct Agent 相比，Plan and Execute 的优势在于先规划后执行，整体目标更清晰，适合复杂的长任务。
