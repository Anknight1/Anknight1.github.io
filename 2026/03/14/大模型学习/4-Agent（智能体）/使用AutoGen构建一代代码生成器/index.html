AutoGen 是微软推出的多 Agent 框架，特别擅长代码生成场景。本文使用 AutoGen 构建一个完整的代码生成器，包含需求分析、代码生成、自动测试和错误修复的完整流程。

AutoGen 简介

AutoGen 的核心是 对话式多 Agent：多个 Agent 通过对话协作完成任务，其中 UserProxyAgent 可以自动执行代码并将结果反馈给 AssistantAgent。

## 总结

AutoGen 通过多 Agent 对话协作，实现了强大的代码生成能力：

- **基础模式**：AssistantAgent + UserProxyAgent，自动执行和修复代码
- **多 Agent 模式**：需求分析、架构设计、编码、审查各司其职
- **自愈模式**：代码执行失败时自动分析错误并修复

AutoGen 特别适合代码生成、数据分析等需要代码执行反馈的场景。
