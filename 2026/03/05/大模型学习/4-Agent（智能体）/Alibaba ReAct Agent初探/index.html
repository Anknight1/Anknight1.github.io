前言

本文是「大模型学习」系列文章之一，持续更新中。

Spring AI Alibaba 是阿里巴巴基于 Spring AI 开发的扩展框架，在提供通义千问等国内模型支持的同时，还引入了图状态机（Graph）来构建更复杂的 Agent。本文介绍其 ReAct Agent 的基本用法。

Spring AI Alibaba 简介

Spring AI Alibaba 的核心特性：

  • 国内模型支持：通义千问（Qwen）、百炼平台等
  • 图状态机：类似 LangGraph 的 StateGraph，支持复杂 Agent 流程
  • 工具生态：集成了搜索、代码执行等常用工具
  • 与 Spring AI 兼容：完全兼容 Spring AI 的接口

总结

Spring AI Alibaba 的 ReAct Agent 在标准 Spring AI 的基础上，增加了图状态机的支持，使得复杂 Agent 流程的构建更加直观。对于需要使用国内模型（通义千问等）的项目，Spring AI Alibaba 是一个很好的选择。
