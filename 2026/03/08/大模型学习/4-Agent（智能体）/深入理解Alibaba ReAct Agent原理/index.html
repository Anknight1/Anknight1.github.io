总结

Spring AI Alibaba 的 ReAct Agent 执行原理：

  1. 图编译：将声明式图定义转换为邻接表结构
  2. 状态机执行：循环执行节点，直到到达 END
  3. 策略合并：通过 Strategy 模式灵活控制状态更新
  4. 工具调用：LLM 返回 ToolCall → ToolNode 执行 → 结果追加到消息历史 → 循环

理解这些原理，有助于在遇到问题时快速定位和解决。
